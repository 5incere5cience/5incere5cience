2. Popper's Criterion of Demarcation
2.1 The Boundary Between Science and Non-Science
Karl Popper (1959) [4] established the fundamental distinction between scientic and
pseudoscientic theories not through their truth content, but through their vulnerability
to refutation:
A theory which is not refutable by any conceivable event is non-scientic.
Irrefutability is not a virtue of a theory (as people often think) but a vice.
For Popper, scientic theories must be

prohibitive they must state clearly which

events cannot happen. The value of a theory lies in its riskiness: the more specic and
bold its predictions, the more opportunities exist for falsication, and thus the more
scientic it is.

2.2 The Falsiability Test
A theory is scientic if:
1. It makes specic, risky predictions.
2. It prohibits certain outcomes from occurring.
3. There exists a Black Swana single observation that would refute it.
A theory is non-scientic (or pseudoscientic) if:
1. It explains all possible observations.
2. Contradictory evidence is reinterpreted as consistent with the theory.
3. It accommodates any outcome through post-hoc rationalization.
